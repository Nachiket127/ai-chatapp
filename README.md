AI-Powered Chat Application


A full-stack, intelligent chat platform featuring real-time communication, AI agent orchestration, and internet-browsing capabilities.

Key Technical Highlights:

AI Agent Orchestration: Integrated LangChain to build an autonomous agent capable of tool usage, specifically leveraging Tavily for real-time internet search capabilities.
Model Integration: Designed the backend to interface with multiple LLMs (Google Gemini/Mistral AI) via a modular service layer.
Full-Stack Architecture: Developed a scalable MERN-stack application with JWT-based authentication, email verification flows, and secure session management.
Real-time Capabilities: Implemented Socket.io to enable bidirectional event-driven communication between the client and server.
State Management: Utilized Redux Toolkit on the frontend for efficient global state handling and optimized data flow.
